## Keyboard shortcuts

### Viewing

| Shortcut  | Action                                |
| --------- | ------------------------------------- |
| Space     | Toggle Mark Up / Mark Down mode       |
| ⌃⌘S or F3 | Show / hide sidebar                   |
| ⌃⌘R       | Toggle readable column                |
| F6 or ⌘L  | Cycle lighting (Auto → Bright → Dark) |


### Zoom

| Shortcut | Action      |
| -------- | ----------- |
| ⌘+       | Zoom in     |
| ⌘-       | Zoom out    |
| ⌘0       | Actual size |


### Find

| Shortcut     | Action         |
| ------------ | -------------- |
| ⌘F, F4, or / | Open Find bar  |
| ⌘G           | Find next      |
| ⌘⇧G          | Find previous  |
| Escape       | Close Find bar |


### File

| Shortcut | Action                                       |
| -------- | -------------------------------------------- |
| ⌘O       | Open…                                        |
| ⌘W       | Close window                                 |
| ⌘R       | Reload                                       |
| ⌘P       | Print…                                       |
| ⌘⇧B      | Open in Browser _(direct distribution only)_ |
